# UI/UX Design Internship Project

## Overview
This repository contains the work completed during my UI/UX Design Internship. Throughout this internship, I worked on multiple design tasks and projects to develop my skills in creating modern, user-friendly, and visually appealing digital interfaces.

The purpose of this internship was to gain practical experience in User Interface (UI) and User Experience (UX) design, while learning how to design solutions that focus on usability, accessibility, and clean visual structure.

## Repository Structure

This repository is organized into weekly folders, where each folder contains the design tasks and projects completed during that specific week of the internship.

Each week includes different UI/UX exercises, layouts, and design concepts that helped in improving practical design skills.

## Skills Practiced

During this internship, I worked on improving the following skills:
User Interface (UI) Design
User Experience (UX) Principles
Layout and Visual Hierarchy
Wireframing and Prototyping
Designing Website Interfaces
Creating Interactive Components

## Tools Used

Figma – UI/UX design and prototyping

Canva – Case study creation and design presentations

## Purpose

The goal of this repository is to document my learning journey and practical experience during the UI/UX internship. It showcases the progression of my design skills through different weekly tasks and projects.
